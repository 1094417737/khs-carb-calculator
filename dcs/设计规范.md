# 设计规范 — KHS耐力补碳计算器

## 设计语言: WorkOutDoors 风格 + Apple Human Interface

参考 WorkOutDoors APP 的数据密集型模块化卡片设计。
- 默认深色模式 (OLED 友好，户外强光可读)
- 模块化数据卡片网格
- 高对比度区域警示色

## 色彩系统

## 组件约定
- 导航栏: 毛玻璃效果 (backdrop-blur-xl) + 深色/浅色自适应
- 开关: iOS 风格 Toggle
- 分段控件: 圆角 pill 按钮组
- 卡片: 深色模式用 border 替代 shadow
- 输入框: 深色/浅色自适应背景
- 数据展示: 大字重数值 + 区域警示色编码

## 响应式断点
```
Mobile:  < 640px   (单列)
Tablet:  640-1023px (两列)
Desktop: >= 1024px  (居中最大880px)
```
