# 技术规格 — KHS耐力补碳计算器

## 技术栈
| 层级 | 技术 |
|------|------|
| 构建 | Vite 5 |
| UI | React 18 + TypeScript 5 |
| 样式 | Tailwind CSS 3 |
| 状态 | React Context + useReducer |
| 持久化 | localStorage |
| 测试 | Vitest + @testing-library/react |

## 目录结构
```
src/
├── types/          # TypeScript 类型定义
│   ├── index.ts    # PlanInputs, StrategyOptions, CalculatorInputs
│   ├── products.ts # GelProduct, DrinkProduct
│   └── results.ts  # CalculationResult 及子类型
├── data/           # 静态数据
│   ├── gels.ts     # 能量胶产品库
│   ├── drinks.ts   # 运动饮料产品库
│   └── constants.ts
├── engine/         # 计算引擎（纯函数）
│   ├── carbs.ts
│   ├── fluid.ts
│   ├── sodium.ts
│   ├── caffeine.ts
│   ├── schedule.ts
│   └── index.ts    # calculate() 编排
├── context/        # React Context 状态管理
├── hooks/          # 自定义 hooks
├── components/
│   ├── layout/     # Header, Footer, Container, Card
│   ├── input/      # 输入表单组件
│   ├── strategy/   # 策略选择组件
│   ├── results/    # 结果展示组件
│   └── ui/         # UI 原语组件
└── utils/          # 工具函数
```

## 数据流
```
用户输入 → dispatch(SET_*) → Context (useReducer)
  → useEffect 触发 engine/calculate()
  → dispatch(SET_RESULTS) → ResultsPanel 渲染
```
