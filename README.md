# KHS耐力补碳计算器

基于运动科学的个性化耐力运动补给方案计算器。参考 Göran Winblad 的自制补给理念：**大多数昂贵的运动补给本质上就是葡萄糖+果糖+电解质+水，用白砂糖+海盐就能极低成本复刻**。

线上地址：**[khs-carb-calculator.pages.dev](https://khs-carb-calculator.pages.dev/)**

## 功能

- **基本信息** — 运动时长、距离配速、心率区间、体重、温度、累计爬升、GI训练水平、出汗率自测
- **补给策略** — 咖啡因方案、自制/商业切换、糖浆比例(白砂糖1:1/2:1/1:0.8)、商业产品选择
- **补给方案** — 碳水需求(g/h+kcal)、补水需求(ml/h)、钠需求(mg/h)、区间范围可视化
- **自制配方** — 葡萄糖/果糖/白砂糖 + 海盐 + 水的精确配比，每瓶计算、成本对比、赛前准备清单
- **容器计算器** — 自定义水壶容量(200-2000ml)，算好每瓶放多少糖多少盐
- **产品对比** — 商业能量胶/运动饮料与目标需求的差额分析
- **咖啡因方案** — 基于体重的赛前+赛中补充时间表
- **补给时间表** — 每30分钟具体动作清单（商业=产品名，自制=克数配比）
- **深色/浅色模式** — 默认深色(户外友好)，一键切换
- **PWA** — 可安装到手机桌面，离线可用
- **一键复制方案** — 纯文本方案分享


## 算法依据

- **碳水**: ACSM 2016 / ISSN 2018 — 心率区间+GI训练+爬升+糖浆比例上限
- **补水**: Sawka 2007 ACSM Position Stand — 温度+体重+强度调节
- **钠**: Maughan & Shirreffs 2012 — 出汗率估算+个体差异+70%补充率
- **咖啡因**: ISSN 2021 — 3mg/kg赛前+1.5mg/kg每2小时
